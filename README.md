### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Rabia-Ghafoor/GlobalNOC.git
    ```

2. Navigate to the Pokemon project directory:

    ```bash
    cd pokemon
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

    ```bash
    npm install --save-dev @types/react @types/react-dom 
    ```
     ```bash
    npm run dev
    ```
